Windows配置Git

Jul 30, 2018 325

首先设置Github的username和emial

然后生成ssh密钥

到 .ssh 文件夹下复制 id_rsa.pub 密钥
路径：C:\Users\god\.ssh (按实际情况寻找)

然后粘贴到Github上面 选择Settings

选择 SSH and GPG keys 然后点击 New SSH key

添加完成后，打开 Git Bash 测试连接

提示 Hi，username 表示成功
